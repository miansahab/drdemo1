# Stronger After 40 landing page

GitHub-ready static website files separated from the original single HTML file.

## Structure

```text
index.html
css/style.css
js/script.js
images/
```

## How to use

Open `index.html` in a browser, or push this folder to GitHub and enable GitHub Pages from the repository settings.

Note: some program photos still load from Pexels CDN. Embedded/base64 images from the original file were extracted into the `images/` folder.
